## Day2 
今天深入探索了机器学习的“无监督”领域，核心心得如下：
K-Means 聚类：掌握了无标签数据的自动归类逻辑。通过迭代优化聚类中心，实现了对复杂样本的群体划分。深刻理解了聚类后的“特征画像”对业务决策的指导意义。
PCA 主成分分析：不仅学习了如何将高维数据投影到低维平面，更理解了其背后的“方差最大化”原则。PCA 是实现高维数据可视化、提升模型训练效率的关键工具。
标准化 (Standardization)：这是无监督学习的“入场券”。理解了在涉及距离敏感型算法时，必须通过标准化消除特征间量纲的影响，确保每个特征在模型中拥有平等的“发言权”。
实战思考：思考了 K-Means 在图像压缩中的应用，以及 PCA + K-Means 在金融反欺诈中的协同作战能力。

## Day5
通过今天的学习 我更加深刻的了解了激活函数的重要性以及CNN的工作原理。
当加入了激活函数后，每一层都能给数据“掰弯”，网路就能画出任意复杂的曲线，拟合任何现实世界的规律。
激活函数的5大核心作用：1.引入非线性 。2.控制输出范围。3.缓解梯度问题。
4.引入稀疏性（例如Rule函数中，当参数小于0，该激活函数会自动将其归化为0，这样在进行后续的反向传播时，会将其ignore掉，模拟了人的大脑）。
5。提升了计算效率。   
在CNN方面，首先知道了CNN是由输入层，卷积层，激活函数层，池化层，批归一化层，Flatten层，全连接层，Dropout层这8个层构成，
主体就三个层，卷积层，池化层，全连接层。在卷积层方面，
首先了解到的是filter（卷积核），它一般是由奇数主导的二维矩阵，然后它要和输入进来的图片数据什么的进行卷积运算，运算完了会得到一个 [（N+2p-F）/S]+1 的一个新的特征矩阵，
然后再经过池化层，将特征的维度降低，缩减模型大小，并且还可以防止过拟合，最后到达全连接层，把所有侦察到的特征汇总起来，经过多轮的正向传播和反向传播，得到最终的结果。
以上就是今天day5对于CNN的学习
